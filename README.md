# Netflix Movies and TV Shows Analysis 

## Project Overview

This project is based on the Netflix Movies and TV Shows dataset. The main purpose of this project is to analyze Netflix content using Python and present the results through an interactive Power BI dashboard.

The project follows a complete data analysis process. It starts with understanding the dataset, then cleaning the data, performing exploratory data analysis (EDA), creating visualizations, and finally building an interactive dashboard in Power BI. This project helped me improve my practical knowledge of Python, data analysis, and dashboard development.

---

## Problem Statement

Netflix has a huge collection of movies and TV shows from different countries, genres, and release years. Raw data is difficult to understand, so it needs to be cleaned and analyzed before meaningful insights can be generated.

The objective of this project is to clean the dataset, analyze the data, identify important trends, and create an interactive dashboard that makes the information easy to understand.

---

## Project Objectives

- Understand the Netflix dataset.
- Clean the dataset.
- Handle missing values.
- Perform Exploratory Data Analysis (EDA).
- Create visualizations using Python.
- Build an interactive Power BI dashboard.
- Generate meaningful business insights.
